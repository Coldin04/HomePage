# Coldin04's Homepage

👋 你好，这是 [Coldin04](https://cold04.com) 的个人主页源码仓库。

这是一个简洁、响应式的静态个人主页，用于展示我的个人信息、社交链接以及一些工作项目。

## ✨ 特性

- **响应式设计**：适配桌面、平板和移动设备。
- **深色模式**：支持系统级深色模式自动切换。
- **纯原生实现**：仅使用 HTML5 和 CSS3，无繁重框架依赖，加载速度快。
- **站点公告系统**：包含顶部横幅和弹窗通知（用于域名迁移等重要通知），支持来源检测。
- **交互细节**：平滑滚动、渐变文字效果、复制功能等。

## 📂 项目结构

```text
.
├── index.html          # 主页入口
├── README.md           # 项目说明
├── aff/                # 推广/联盟链接页面
├── pgp/                # PGP 公钥展示页面
└── src/                # 静态资源目录
    ├── style.css       # 主要样式表
    ├── style-page.css  # 其他页面样式
    ├── favicon.ico     # 网站图标
    └── avatar.webp     # 头像图片
```
